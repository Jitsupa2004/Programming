# Programming

## Keyword
**Programming**

---

## Definition

### English (Source 1 – Oxford Languages / Google Dictionary)
> "Programming is the process of designing and building an executable computer program to accomplish a specific computing result or to perform a specific task, such as processing data, solving problems, or automating repetitive tasks."

### English (Source 2 – Techopedia)
> "Programming is the implementation of logic to facilitate specified computing operations and functionality. It occurs in one or more languages, which differ by application, domain and programming model."

### Thai (สรุปในแบบของตัวเอง)
Programming หรือ "การเขียนโปรแกรม" คือการบอกให้คอมพิวเตอร์ทำงานตามที่เราต้องการ โดยใช้ภาษาที่คอมเข้าใจได้ — เหมือนการเขียนสูตรอาหารให้หุ่นยนต์ทำตาม ถ้าสูตรผิด ก็พัง 😅

## Explanation (ขยายความ)

### Programming คืออะไร?
Programming คือกระบวนการเขียนชุดคำสั่ง (Code) ให้คอมพิวเตอร์ทำงานตามที่กำหนด โดยใช้ภาษาโปรแกรมมิ่ง (Programming Language) เป็นสื่อกลางระหว่างมนุษย์กับเครื่องจักร

### 🧩 ขั้นตอนการเขียนโปรแกรมโดยทั่วไป
1. **วิเคราะห์ปัญหา** — รู้ว่าต้องการให้โปรแกรมทำอะไร
2. **ออกแบบ Algorithm** — วางแผนลำดับขั้นตอนการทำงาน
3. **เขียน Code** — แปลงแผนเป็นภาษาโปรแกรม
4. **ทดสอบ (Testing)** — ตรวจสอบว่าทำงานถูกต้อง
5. **แก้ Bug** — แก้ไขส่วนที่ผิดพลาด

### 💻 ภาษาโปรแกรมยอดนิยม

| ภาษา | ใช้ทำอะไร |
|------|---------|
| Python | Data Science, AI, Automation |
| JavaScript | Web Development (Front-end & Back-end) |
| Java | Mobile App (Android), Enterprise |
| C/C++ | System Programming, Game Engine |
| SQL | จัดการฐานข้อมูล |

### ทำไม Programming ถึงสำคัญ?
- เป็นทักษะพื้นฐานของอาชีพด้าน IT ทุกสาย
- ช่วยแก้ปัญหาและทำให้งานซ้ำ ๆ เป็นอัตโนมัติ
- เป็นรากฐานของทุกซอฟต์แวร์ แอปพลิเคชัน และระบบดิจิทัล

### ตัวอย่าง Code เบื้องต้น (Python)
```python
# สวัสดีโลก!
name = "จิตรสุภา"
print(f"สวัสดีค่ะ คุณ {name} 👋")

# ลูปง่าย ๆ
for i in range(1, 6):
    print(f"นับ {i}")
```
